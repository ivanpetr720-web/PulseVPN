# PulseVPN — Android Studio Project

## Как собрать APK (нужен ПК):

1. Скачай Android Studio: https://developer.android.com/studio
2. Распакуй этот ZIP
3. File → Open → выбери папку PulseVPNApp
4. Подожди пока скачаются зависимости (1-2 мин)
5. Build → Build Bundle(s)/APK(s) → Build APK(s)
6. APK будет в: app/build/outputs/apk/debug/app-debug.apk

## Что в приложении:

### Главный экран (Защита)
- Кнопка Connect с анимацией пульса
- РЕАЛЬНЫЙ IP через ipapi.co (не фейковый!)
- Флаг и страна определяются автоматически
- Статистика DL/UL при подключении
- Таймер сессии
- Переключение WireGuard / OpenVPN / Auto

### Вкладка Серверы
- 3 БЕСПЛАТНЫХ сервера (🆓 Германия, Нидерланды, Швеция)
- 9 PREMIUM серверов (заблокированы до покупки)
- Поиск по стране/городу
- Пинг с цветовой индикацией (зелёный/жёлтый/красный)

### Вкладка Аккаунт
- Информация о бесплатном плане
- Карточка "Купить Premium" за 299₽/мес
- Kill Switch, Stealth Mode, Блокировка рекламы
